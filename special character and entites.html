Special Characters and HTML Entities

In HTML, some characters cannot be used directly, so we use entities.

Common HTML Entities:

Less than symbol: < (Code: &lt;)

Greater than symbol: > (Code: &gt;)

Ampersand symbol: & (Code: &amp;)

Double quote: " (Code: &quot;)

Single quote / Apostrophe: ' (Code: &apos;)

Copyright symbol: © (Code: &copy;)

Registered symbol: ® (Code: &reg;)

Trademark symbol: ™ (Code: &trade;)

Rupee symbol: ₹ (Code: &#8377;)

Example Usage:

This website is © 2025. All rights reserved.

Price: ₹ 250 only.
